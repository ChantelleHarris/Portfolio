Static Layout

Static Layout really taught me how to position things. This is where I had exposure to CSS Grid, Flexbox, and a little animation. I also really enjoyed learning about navigation bars and links, code structure, assigning image Urls to the background of a page, and using assets in a project.

This project served as both a learning tool and confidence booster. Click the image to see more on GitHbub; I find it quite enjoyable to deploy to GitHub.

API Challenge One

Studio Ghibli API

I am a woman who loves challenges, and API Challenge One proved to be an excellent one. With it, I explored video embedding, background audio, and grabbing and displaying data from an API using Java Script and DOM Manipulation.

I think this challenge served as a catalyst for 'belief' in myself as a future web developer. There were sleepless nights accompanied with inevitable mental exhaustion...and it all worked out. It all came together--"quite a feat," I'd say; I am proud.

Even if you're not a fan of Studio Ghibli as I am, I hope you enjoy my work. Click to see more; this project is deployed to Firebase.
